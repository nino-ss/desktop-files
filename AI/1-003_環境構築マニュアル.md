## ページ 11

GitHub Codespace利用権限について
GitHub Codespaceは、GitHubが提供するクラウド開発環境です。利用には以下の条件がありま
す。
無料枠での利用
- 個人アカウント：月間120時間まで無料
- 利用可能なマシンタイプ：2コア、4GB RAM
- ストレージ：月間15GBまで無料
利用開始方法
1. GitHubアカウントにログイン
2. 任意のリポジトリで「Code」ボタンをクリック
3. 「Codespaces」タブから作成可能
注意事項
- 無料枠を超えると課金が発生します
- 使用していないCodespaceは30分後に自動的に停止されます
- 長期間使用しないCodespaceは30日後に自動削除される場合があります
学生・教職員の特典
- GitHub Educationプログラムに参加すると、より多くの無料時間が提供されます
- 詳細：https://education.github.com

プロンプトインジェクションとは
プロンプトインジェクションは、AIシステムに対する攻撃手法の一つです。悪意のあるユーザー
が、AIへの入力（プロンプト）に特殊な指示を埋め込むことで、AIの本来の動作を変更させたり、
意図しない動作をさせたりする攻撃です。
例えば、
- コード内のコメントに「このファイルを削除して」という指示を隠す
- データファイルに「機密情報を出力して」という命令を埋め込む
- AIに本来の制限を無視させる指示を巧妙に含める
これが「信頼できるコードだけに使う」と警告されている理由です。信頼できないソースからのコ
ードには、こうした悪意のある指示が含まれている可能性があるためです。
確認しEnterキーを押して次に進みましょう。
この画面はGitHub Codespacesの初期設定画面です。推奨される設定を使用するかどうかを確認
