SQL（Structured Query Language）は、データベースを操作するための専用言語です。データ
ベースに対してデータの作成、読み取り、更新、削除などの操作を行うために使用されます。

-
CREATE TABLE users
→ usersという名前のテーブル（データの入れ物）を作成
-
INSERT INTO tasks
→ tasksテーブルにデータを追加
-
SELECT * FROM comments
→ commentsテーブルからすべてのデータを取得
