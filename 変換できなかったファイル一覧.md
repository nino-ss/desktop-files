## 対処方法

### 方法1: 手動でPDFまたはWordにエクスポート

1. Pagesアプリでファイルを開く
2. 「ファイル」→「書き出す」→「PDF」または「Word」を選択
3. エクスポートしたファイルを同じ場所に保存
4. 変換スクリプトを再実行

### 方法2: 個別に変換

重要なファイルのみ、上記の方法でエクスポートしてから変換スクリプトを実行してください。

### 方法3: Automatorを使用した一括エクスポート

macOSのAutomatorを使用して、複数のPagesファイルを一括でPDFまたはWordにエクスポートすることもできます。

## 変換スクリプト再実行方法

cd /Users/ninomiya/Desktop

python convert_to_markdown.py [エクスポートしたPDF/Wordファイルのパス]
