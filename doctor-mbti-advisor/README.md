# ドクターMBTI診断アプリ

MBTI（Myers-Briggs Type Indicator）理論を医師向けにアレンジした診断アプリケーションです。

## 概要

このアプリは、医師の性格タイプを診断し、以下の情報を提供します：

- **MBTIタイプ判定**: 16のMBTIタイプのいずれかを判定
- **医師としての特徴**: あなたの診療スタイルや性格的特徴
- **得意なこと**: あなたの強みと活かせる医療分野
- **苦手なこと**: 注意すべき弱点
- **陥りやすい失敗**: よくある失敗パターン
- **アドバイス**: 改善のための具体的なアドバイス

## 使い方

1. `index.html`をブラウザで開く
2. 「診断を開始する」ボタンをクリック
3. 12問の質問に直感的に答える
4. 結果を確認

## 診断内容

### 測定する4つの軸

1. **E（外向）vs I（内向）**: エネルギーの向け方
2. **S（感覚）vs N（直観）**: 情報の受け取り方
3. **T（思考）vs F（感情）**: 意思決定の方法
4. **J（判断）vs P（知覚）**: 生活スタイル

### 16のMBTIタイプ

各タイプには医師としての以下の特性が記載されています：

- ISTJ: 責任感の強い実践派ドクター
- ISFJ: 献身的なケアラードクター
- INFJ: 理想を追求するヒーラードクター
- INTJ: 戦略的思考の革新派ドクター
- ISTP: 冷静沈着な問題解決ドクター
- ISFP: 共感力の高い芸術派ドクター
- INFP: 理想主義の共感派ドクター
- INTP: 理論追求の分析派ドクター
- ESTP: 行動派のアクティブドクター
- ESFP: エンターテイナードクター
- ENFP: 情熱的な革新派ドクター
- ENTP: 挑戦的なディベータードクター
- ESTJ: リーダーシップの組織派ドクター
- ESFJ: 温かいサポータードクター
- ENFJ: カリスマ的なメンタードクター
- ENTJ: 戦略的なリーダードクター
